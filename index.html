Google AdWords Keyword Tool

About the tool

I have created a simple program to help you wrap your keywords phrases in 'quotation marks' and [square brackets]. I hope it will help you with your work on campagins.

How to use keyword tool

  1. Type or paste all your keywords to first table. Remember to put one keyword per row. Dont worry about any free spaces between the rows and spaces at the end of words. Program will remove them. them.
  2. Click button generate.
  3. Copy selected type of match you need. If you want, You can use for that buttons that are loacted under each table
